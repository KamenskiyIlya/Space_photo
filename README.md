# Скачивание фото из NASA и SpaceX

Этот набор скриптов позволяет скачивать фото космоса, Земли, запуска ракет из NASA и SpaceX и сохранять их к Вам на компьютер.

## Что необходимо подготовить

Для работы скриптов будет необходима установка Python 3. После чего нужно подгрузить необходимые пакеты для нормальной работы скрипта с помощью pip. Для этого введите в командной строке команду:
```
pip install -r requirements.txt
```

Если ВЫ планируете делать много запросов на сайты NASA, тогда стоит у них завести собственный API ключ, т.к. по демо ключу - `DEMO_KEY`, который указан по умолчанию, кол-во запросов в час ограничено. Для этого нужно зайти на их [сайт](https://api.nasa.gov), ввести там свои данные и сгенерировать свой токен.

*фото*

Для дальнейшего использования токена при запросах в скриптах `get_NASA_APOD_image.py` и `get_NASA_EPIC_image.py`, необходимо создать переменное окружение `.env`, в котором необходимо прописать Ваш ключ в слудующем формате:
```
NASA_API_KEY='Ваш API ключ'
```
Дальше скрипты сами будут подтягивать Ваш ключ при исполнении.

Также в репозитории имеются 3 скрипта, которые осуществляют публикации в телеграмм канал при помощи бота. Для работы этих скриптов необходимо в переменном окружении `.env` указать ключб бота в параметр `TG_BOT_TOKEN` и id чата в параметре `CHAT_ID`.

## Как пользоваться

Данный репозиторий предлагает такой список скриптов:
- `get_SpaceX_launch_images.py`
- `get_NASA_APOD_image.py`
- `get_NASA_EPIC_image.py`
- `publish_text_in_tg.py`
- `publish_photo_in_tg.py`
- `auto_publisher.py`

Каждый из них при запуске создает в директории откуда запускается папку `images`, если она не была создана до этого.

### get_SpaceX_launch_images

Данный скрипт помогает скачать фото запуска ракет с сайта SpaceX. При вызове пользователь может указать следующий параметр
-id - id запуска ракеты.

Эти параметр не обязательно указывать, без него скрипт скачает фото самого последнего запуска. Если же Вы хотите получить фото случайного запуска, тогда укажите в этом параметре слово `random`. Для более подробного описания можно вызвать команду в терминале:
```
python get_SpaceX_launch_images.py -h
```
Вызов этого скрипта осуществялется следующим образом:
```
python get_SpaceX_launch_images.py
```
С указанием параметров:
```
python get_SpaceX_launch_images.py -id=*id запуска*
```
Для того чтобы скачать фото случайного запуска ввести следующую команду
```
python get_SpaceX_launch_images.py -id=random
```

### get_NASA_APOD_image

Следующий скрипт позволит Вам скачать так называемое [APOD(Astronomy Picture of the Day)](https://api.nasa.gov/#apod) в любом кол-ве при помощи API сайта NASA. Этот скрипт имеет только один параметр:
- -c - кол-во фото, которое нужно скачать за раз. Если его не указывать, будет скачиваться только 1 фото.

Вызывается этот скрипт следующей командой:
```
python get_NASA_APOD_image.py
```
C указанием параметра:
```
python get_NASA_APOD_image.py -c=10
```
Для подробного описания скрипта ввести в терминале
```
python get_NASA_APOD_image.py -h
```

### get_NASA_EPIC_image

Данный скрипт позволит Вам скачать настоящие фото Земли из API сервиса NASA под названием [EPIC(Earth Polychromatic Imaging Camera)](https://api.nasa.gov/#epic). Этот скрипт имеет только один параметр:

-c - кол-во фото, которое нужно скачать за раз. Если его не указывать, будет скачиваться только 1 фото. Однако за раз не может быть скачано больше 13 фото.

Вызывается этот скрипт следующей командой:
```
python get_NASA_EPIC_image.py
```
C указанием параметра:
```
python get_EPIC_APOD_image.py -c=10
```
Для подробного описания скрипта ввести в терминале
```
python get_NASA_APOD_image.py -h
```

### publish_text_in_tg
Это простой скрипт для постинга ботом какого-либо сообщения в группе. От Вас потребуется ввести эту команду в терминал:
```
python publish_text_in_tg.py
```
После чего ввести Ваше сообщение.

### publish_photo_in_tg

Этот скрипт позволяет запостить фото из Вашей папки `images`, в которую Вы до этого загружали фото с помощью других скриптов, в группу телеграма. Этот скрипт имеет один параметр:
- n - имя файла, который Вы хотите запостить, указывать необходимо вместе с его расширением. Если не указывать этот параметр будет выбрано случайно фото из папки `images`.

Базовый вызов этого скрипта осуществляется с помощью команды:
```
python publish_photo_in_tg.py
```
Пример вызова с параметром:
```
python publish_photo_in_tg.py -n=nasa_apod_1.jpg
```
Вызов подробной информации по скрипту:
```
python publish_photo_in_tg.py -h
```

### auto_publisher

Этот скрипт позволяет Вам в автоматическом режиме постить фото из папки `images` в телеграм канал. Стандартно фото будет поститься раз в 4 часа, пока запущен скрипт. Вы можете настроить время постинга указав в переменном окружении `.env` промежуток между публикациями при помощи параметра DELAY_TIME. Время в DELAY_TIME указывается в секундах. Поэтому не забывайте перевести часы и минуты в секунды.

скрипт вызывается следующей командой:
```
python auto_publisher.py
```
